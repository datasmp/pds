DataFrames#

A DataFrame is a two-dimensional data structure characterized by labeled axes (rows and columns).

  • Resembling a Microsoft Excel worksheet, it accommodates mixed data types efficiently.

  • Primarily utilized for data wrangling and manipulation.

  • DataFrames offer numerous methods for tasks such as data cleaning, exploration, and preprocessing.

  • They support vectorized operations across rows or columns and facilitate visualization.

  • The main data structure of the pandas library, the DataFrame, requires importing the library to create and manipulate dataframes and utilize its methods.

  • It’s commonplace to abbreviate pandas as pd for brevity in code.

    • import pandas as pd

Create Dataframes#

import pandas as pd

Use Tuples or Lists#

  • Use pd.DataFrame()

  • Column and row labels can be provided. If they ar not given then default values \(0,1,2,...\) are ised.

state_tuple = ('NY', 'OK', 'TX')
df = pd.DataFrame(state_tuple)
display(df)

1 OK
2 TX 
state_list = ['NY', 'OK', 'TX']
df = pd.DataFrame(state_tuple)
display(df)

2 TX

  • You can provide the the row and column labels using the index and columns parameters.

state_list = ['NY', 'OK', 'TX']
df = pd.DataFrame(state_tuple, index=['John', 'Liz', 'Tim'], columns=['State'])
display(df)
State

Liz OK
Tim TX

Slicing#

Slicing can be done either by using the row and column indexes or labels.

  1. Row and column indexes: Use iloc[row index, column index]

  2. Row and column labels : Use loc[row label, column label]

Let’s remember the small dataframe df_sh.

df_sh
State Height
John NY 160
Liz OK 180
Tim TX 140

  • Let’s try to access the height of Tim which is 140 by using loc and loc.

    • row label of 140 is Tim.

    • column label of 140 is Height.

df_sh.loc['Tim', 'Height']

140

  • Let’s try to access the height of Tim which is 140 by usin iloc.

    • row index of 140 is index of Tim in [‘John’, ‘Liz’, ‘Tim’] which is 2.

    • column index of 140 is index of Height in [‘State’, ‘Height’] which is 1.

df_sh.iloc[2,1]

140

  • You can also do slicing as in strings, tuples, and lists.

  • Slicing with loc (labels) has a small difference.

    • loc is inclusive for the end label which means the end label is included.

  • Example: The column labels of df_sh dataframe [‘State’, ‘Height’]

    • df_sh.loc[‘Liz’:,:’Height’] inludes he columns up to and including ‘Height’.

# rows starting from index 1 (includes 1): row indexes: 1, 2
# columns up to index 1 (does not include 1): column index: 0

df_sh.iloc[1:,:1]
State
Liz OK
Tim TX 
# rows starting from  'Liz' (includes 'Liz'): row labels: 1, 2
# columns up to label 'Height' (includes 'Height'): column label: 'State', 'Height'

df_sh.loc['Liz':,:'Height']
State Height
Liz OK 180
Tim TX 140

Vectorized Operations#

Vectorized operations can be done using the columns of a dataframe which means with a single algebraic operation like + operation can be performed for each row.

df_grades.head()

0 mqtvy 37047871 10 M 30 91 69 93 17 50
1 jbbsx 35439616 11 F 6 18 93 9 98 91
2 mrvab 35543247 11 M 78 92 60 43 34 26
3 bjyve 61282135 9 M 60 8 10 99 80 87
4 rlpsr 53448034 10 M 3 38 45 43 79 69

    • range(df.shape[1]) for column indexes

Saving Dataframes#

Google Colab#

  1. You need to do mounting to make the connection between your Google Drive and Google Colab notebook using the following code.

from google.colab import drive
drive.mount('/content/drive')

  1. You can save your dataframe as an excel or csv file using the path of the folder where you want to save your dataframe.

  df_sh.to_excel('/content/drive/My Drive/data_files/state_and_height.xlsx')

  • In the code above data_files is a folder in My Drive

  • df_sh dataframe is saved as an excel file in data_files folder and the name of the excel file is state_and_height.

Jupyter Notebook#

The following codes will save your dataframe as an excel or csv file inside the folder where your Jupyter notebook is. You can use the same codes also with other editors including Spider and Pycharm.

  • Use name of dataframe.to_excel(‘name of the excel file.xlsx’)

  • Use name of dataframe.to_csv(‘name of the csv file.csv’)

# save as an excel file
df_sh.to_excel('state_and_height.xlsx')

#save as a csv file
df_sh.to_csv('state_and_height.csv')

  • If you want to save your dataframe in a different folder, you need to provide the path.

  • The following code saves the dataframe as a csv file in a folder called data_files

df_sh.to_csv('data_files/state_and_height.csv')
